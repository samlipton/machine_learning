# Machine Learning 

This repository contains a collection of machine learning projects solved using modular workflows. 

The directory structure is lightweight: 
- **data/**: storage for raw, cleaned and processed datasets
- **scripts/**: reusable building blocks implementing core functionalities
- **projects/**: workflow-oriented modules grouped around a machine learning theme
- **notebooks/**: documentation and explotary analysis on machine learning topics

The objectives are ordered as follow: 
1. apply multiple machine learning models
2. compare model performance across datasets
3. optimize preprocessing and hyperparameters
4. build reproducible pipelines
